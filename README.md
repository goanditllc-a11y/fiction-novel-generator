## ⬇️ Download & Install

1. Go to the [**Latest Release**](https://github.com/goanditllc-a11y/fiction-novel-generator/releases/latest)
2. Download **`FictionNovelEditor-Windows-x64.zip`**
3. Extract the ZIP to any folder
4. Double-click **`FictionNovelEditor.exe`** to launch
5. (Optional) Right-click the `.exe` → **Send to** → **Desktop (create shortcut)** for a desktop icon

> **Note:** On first launch, the app will download a small AI model (~500MB). This only happens once.
